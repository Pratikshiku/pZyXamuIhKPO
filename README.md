**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 商品信息
5. 商品资讯
6. 个人中心
7. 后台管理
8. 购物车
9. 客服
10. 商品评价管理
11. 我的收藏管理
12. 订单管理


**管理员功能：**
1. 登录
2. 个人中心
3. 用户管理
4. 商品分类管理
5. 颜色管理
6. 商品信息管理

9. 订单管理
